Problem 

Students forget tasks and feel overwhelmed; existing apps are cluttered. 

Approach 

Built a token-based design system first, then assembled 7 screens from reusable 
components. 

Craft 

8-point spacing grid, WCAG-AA contrast, smart-animate prototype with a bottom-sheet 
overlay.

Outcome

A tappable prototype + a reusable component library. https://www.figma.com/proto/qPOnoXwvZt2sHxO2shQJY3/TIDE?node-id=5-3&t=D9sP06Vo9n2KczHm-1
